### **1. Universal School Choice**

**Policy**: Every child receives $15,000/year (inflation-adjusted) to attend any accredited public, private, charter, or mentor-led program. Funds follow the child. No geographic monopoly.

|**HLG Alignment**|**Reasoning & Evidence**|**Circuit-Breaker**|
|---|---|---|
|**1. Protect**|Failing schools correlate with higher crime (Chicago: dropout → 4x arrest risk).|Auto-expand if violent incidents in failing districts ↓10% in 2 years.|
|**2. Fair Shot**|Zip code currently predicts 70% of outcomes (Chetty). Vouchers equalize inputs.|Mandatory if test-score gap >15% between richest/poorest districts.|
|**3. One Rulebook**|Same $15k for every child — no income, race, or disability carve-outs.|N/A (universal).|
|**4. Evidence**|FL (2000–2025): +12% reading, +15% math; no cost increase (FL DOE).|Sunset if national average scores flatline 3 years.|
|**5. Liberty**|Parents opt out of failing systems.|Annual parent exit survey; <60% satisfaction → program audit.|
